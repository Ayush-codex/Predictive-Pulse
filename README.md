# Hypertension Prediction AI

AI-powered healthcare application that predicts hypertension risk using Machine Learning and Flask.

## Features

- Machine learning prediction using Logistic Regression
- Flask web application
- Interactive medical dashboard
- Clinical recommendations system

## Tech Stack

Python  
Flask  
Scikit-learn  
HTML / CSS  
NumPy  
Joblib  

## How to Run

```bash
pip install flask scikit-learn numpy joblib
python app.py
